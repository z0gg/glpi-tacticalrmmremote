# Contributing

Thanks for helping improve the plugin.

## Rules

- Keep the plugin folder name as `tacticalrmmremote`
- Keep changes compatible with supported GLPI versions
- Update `CHANGELOG.md` for user-visible changes
- Test on a non-production GLPI before merging

## Pull requests

- One feature or fix per PR
- Explain the GLPI version tested
- Add screenshots when UI changes
- Keep release archives out of git history

## Checklist

- [ ] Code tested on GLPI
- [ ] Changelog updated
- [ ] README updated if needed
- [ ] No generated archives committed
